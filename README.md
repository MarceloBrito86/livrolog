# 📚 LivroLog

PWA completo para registrar e acompanhar sua jornada de leitura — tema escuro, capas automáticas e estatísticas detalhadas.

## Funcionalidades

- 📖 **Físico / 📱 Digital** — organize por formato de leitura
- 🔍 **Busca automática de capa** via Google Books API
- ⭐ **Avaliação de 0 a 10** com sistema de estrelas
- 👍👎 **Gostei / Não gostei / Neutro**
- 🏷️ **16 gêneros literários** (Ficção, Romance, Fantasia, Biografia, Autoajuda, HQ, etc.)
- ✍️ **Autor, ano e número de páginas**
- 📝 **Anotações** e citações marcantes
- 🏆 **Ranking** dos livros mais bem avaliados, com medalhas
- 📊 **Estatísticas**: páginas lidas, média por livro, top autores, top gêneros, distribuição de notas
- 🔎 Busca por título ou autor, filtros por gênero e ano
- 💾 **Funciona offline** — dados salvos localmente no dispositivo
- 📤📥 **Backup e restauração** via arquivo .json
- 📱 **Instalável** como app no iPhone/Android (PWA)

## Como instalar no celular

1. Acesse o site publicado pelo GitHub Pages
2. No iPhone: abra no **Safari** → toque em **Compartilhar** → **Adicionar à Tela de Início**
3. No Android: abra no **Chrome** → menu **⋮** → **Adicionar à tela inicial**

## Tecnologia

HTML, CSS e JavaScript puros — sem frameworks, sem servidor, sem banco de dados externo. Os dados ficam salvos no `localStorage` do próprio dispositivo, garantindo privacidade total.
